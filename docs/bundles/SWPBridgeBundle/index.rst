BridgeBundle
============

This bundle provides tools which help you to integrate `Superdesk`_ data with Superdesk Web Publisher.

:doc:`/components/Bridge/index`, which is used by this bundle, provides a generic interface to create different type of validators, data transformers, and models, which in turn help to pull and process data from Superdesk.
